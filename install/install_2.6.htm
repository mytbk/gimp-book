安装软件

这里下载的是 exe 可执行文件，向 windows xp 系统里安装。


双击此图标，程序是从 www.gimp.org 下载的。


开始运行 windows 下的安装程序，这个 exe 文件可安装到 windows 2000 和 winodows xp 上。点击“Next”


版权说明，这里用到的是 GPL 2.0 版，大部分自由软件所用的。点击“Next”


选择安装方法，精典安装“Install now”很方便，适合初级用户。而我选择的是“Customize”。点击“Next”


修改安装目录，我要安装到 D 盘上。点击“Next”


一些提示信息，看看就行，讲的是安装了什么，不用动。点击“Next”


一样不用动，这里是设置文件关联的。点击“Next”


开始菜单的文件夹的名字，不用改了。点击“Next”


设置启动时的选项位置，不用改了。点击“Next”


显示安装设置信息，如有不对可以改，需要后退，这里不用。点击“Next”


正进行安装界面。等着就好。


安装完成，是否运行 GIMP 。点击“Finish”


这是安装完成后，启动软件的界面。
