新建图层
  创建新图层并将它添加到图像。
  快捷键：Shift + Ctrl + N
从可见项创建
  根据可见项创建新图层。复制最顶上的可见图层创建新图层，如果所有图层不可见，则创建一透明背景的图层。
复制图层
  创建图层的副本，并将其添加到图像。
固定图层
  固定浮动图层。
向下合并
  将此图层与其下的一层合并。
删除图层
  删除此图层。
堆栈
  
  选择前一图层
    选择活动图层上方的图层。
  选择后一图层
    选择活动图层下方的图层。
  选择顶端图层
    选择最顶端图层
  选择底端图层
    选择最底端图层
  升高图层
    使此图层上升一步。
  降低图层
    使此图层下降一步。
  图层升至顶端
    使此图层升至最顶端。
  图层降至低端
    使此图层升至最低端。
  反转图层顺序
    反转图像中图层顺序。
蒙板
  
  添加图层蒙板
    添加蒙板以允许对透明度的非破坏性编辑。如图
    
  应用图层蒙板
    应用图层蒙板，并删除它。
  删除图层蒙板
    删除图层蒙板。
  显示图层蒙板
    显示图层蒙板。开关选项。
  编辑图层蒙板
    在图层蒙板上工作，开关选项。
  禁用图层蒙板
    禁用图层蒙板效果。开关选项。
  蒙板到选区
    用图层蒙板代替选区。通过图层蒙板，再通过此功能可选择出选区。
  添加到选区
    将图层蒙板添加到当前选区。
  从选区中减去
    当前选区与图层蒙板的交集。
  与选区相交
    取得选区减去图层蒙板的集。
透明
  
  添加 Alpha 通道
    添加透明度信息到图层。
  删除 Alpha 通道
    删除图层的透明度信息。
  Alpha 阈值
    令透明度满或空，如图
    
  半平整
    以当前背景色部分替换透明度。
  颜色到 Alpha
    将指定的颜色转换为透明。
  Alpha 到选区
    用图层的 Alpha 通道替换选区。
  添加到选区
    将图层 Alpha 通道添加到当前选区。
  从选区中减去
    从当前选区减去图层的 Alpha 通道。
  与选区相交
    图层的 Alpha 通道与选区的交集。
变换
  
  水平翻转
    即镜像。像照镜子一样的效果。
  垂直翻转
    垂直方向的镜像。
  顺时针旋转90度
    向右手边旋转90度。
  逆时针旋转90度
    向左手边旋转90度。
  旋转180度
    不分左右，呵呵，旋转180度，效果是一样的。
  任意旋转
    任意角度旋转，如图
    
  位移
    像素偏移，在图像边缘处可以设置成循环。
    
图层边界大小
  调整图层边界大小。
图层到图像大小
  将图层的大小设置成图像的尺寸。
缩放图层
  改变此图层内容的大小。
剪栽到选区
  剪栽图层到选区的范围。
自动剪栽图层
  移动图层中，空（没有内容）的边缘。
